# PetOn (반려동물 종합 플랫폼)

## 📌 프로젝트 소개
반려동물 관련 정보 제공 및 커뮤니티 기능을 제공하는 웹 서비스입니다.
반려동물 양육 과정에서 정보 탐색, 커뮤니티 활동, 병원 상담, 쇼핑 등 
다양한 서비스를 각각 이용해야 하는 불편함을 해결하고자 기획한 통합 웹 플랫폼 프로젝트입니다.
회원 관리, 반려동물 관리, 게시판, 쇼핑몰 기능을 구현하여 사용자 편의성과 접근성을 높이는 데 중점을 두었습니다.
